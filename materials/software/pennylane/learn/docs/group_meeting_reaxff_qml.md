# 量子机器学习 (QML) 组会汇报大纲
## 针对 ReaxFF-DFT 残差学习的原理科普

---

### 第 1 页：为什么选择量子机器学习 (QML)？
**核心优势：指数级特征空间与小样本表达能力**

*   **指数级计算空间**：
    *   $n$ 个量子比特可以表示 $2^n$ 维的状态向量。
    *   **对比**：对于 20 个分子特征，经典空间是 20 维，而量子希尔伯特空间可达 $2^{20} \approx 100$ 万维。
*   **天然的非线性升维工具**：
    *   QML 本质上是一种极其高效的**非线性升维方法**。它能将低维的分子构型特征映射到极高维的量子态空间，从而更容易捕捉复杂的残差规律。
*   **小样本优势**：
    *   DFT 计算昂贵导致样本量小。QML 的高维投影能力使其在泛化性能和数据效率上具有超越经典神经网络（如 GNN）的潜力。

---

### 第 2 页：量子机器学习的数学表示
**核心原理：量子态即向量，量子操作即矩阵**

*   **量子比特的向量表示**：
    *   单量子比特状态存在于二维复向量空间 $\mathbb{C}^2$ 中。
    *   **计算基态**：$|0\rangle = \begin{bmatrix} 1 \\ 0 \end{bmatrix}$，$|1\rangle = \begin{bmatrix} 0 \\ 1 \end{bmatrix}$。
*   **叠加原理 (Superposition)**：
    *   量子态可以表示为基态的线性组合：$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle = \begin{bmatrix} \alpha \\ \beta \end{bmatrix}$。
    *   **归一化条件**：$|\alpha|^2 + |\beta|^2 = 1$（确保测量概率总和为 1）。
*   **Dirac 记号 (Dirac Notation)**：
    *   **右矢 (Ket)** $|\psi\rangle$：表示列向量。
    *   **左矢 (Bra)** $\langle\psi|$：表示其共轭转置的行向量。

---

### 第 3 页：量子数据编码 (Encoding)
**核心原理：将经典数据转化为量子态的“大门”**

*   **Angle Embedding（角度编码）**：
    *   **方法**：将特征值 $x_i$ 映射为旋转门的旋转角度。
    *   **数学形式**：$|\psi(x)\rangle = \bigotimes_{i=1}^{n} R_Y(x_i) |0\rangle$。
    *   **特点**：最常用，资源消耗低（$n$ 个特征只需 $n$ 个比特）。
*   **Amplitude Embedding（振幅编码）**：
    *   **方法**：将整个特征向量 $x$ 编码为量子态的振幅系数。
    *   **数学形式**：$|\psi(x)\rangle = \sum_{i=1}^{2^n} x_i |i\rangle$。
    *   **特点**：表达密度极高，适合大规模特征。
*   **预处理**：数据必须缩放到 $[0, \pi]$，以匹配旋转门的周期性。

---

### 第 4 页：量子门电路 (Quantum Gates)
**核心原理：实现非线性关联与逻辑变换**

*   **单比特旋转门（非线性来源）**：
    *   $R_Y(\theta) = \begin{pmatrix} \cos(\theta/2) & -\sin(\theta/2) \\ \sin(\theta/2) & \cos(\theta/2) \end{pmatrix}$
    *   $R_Z(\theta) = \begin{pmatrix} e^{-i\theta/2} & 0 \\ 0 & e^{i\theta/2} \end{pmatrix}$
    *   **作用**：通过三角函数引入初始非线性映射。
*   **双比特纠缠门（关键机制）**：
    *   **CNOT 门**：如果控制位为 1，则翻转目标位。
    *   **非线性关联**：创建量子比特间的不可分离态（$|\psi\rangle \neq |\psi_1\rangle \otimes |\psi_2\rangle$）。
    *   **作用**：捕获分子特征之间（如原子对交互）的高阶非经典关联。

---

### 第 5 页：变分量子电路 (VQC) 与测量
**核心原理：经典-量子混合优化架构**

*   **电路设计 (Ansatz)**：
    *   由**编码层**（输入数据）和**变分层**（可训练参数 $\theta$）组成。
    *   公式：$|\phi(x, \theta)\rangle = U(\theta) U_{enc}(x) |0\rangle$。
*   **量子测量 (Measurement)**：
    *   测量 Pauli-Z 算符的期望值：$\langle Z \rangle = \langle \phi | Z | \phi \rangle$。
    *   **Born 规则**：测量概率 $p(m) = |\langle m|\phi\rangle|^2$。模平方项引入了关键的非线性。
*   **混合优化循环**：
    1.  **量子端**：执行电路，测量输出预测值 $\hat{y}$。
    2.  **经典端**：计算 Loss，通过 **Parameter-shift rule** 计算梯度并更新 $\theta$。
    3.  **循环**：直至预测残差 $\Delta$ 与 DFT 真值匹配。

# 第 2 页：量子机器学习核心原理

## 🌊 QML 工作流程：编码 → 演化 → 测量

```
       经典数据 x              量子态 |ψ(x)⟩           演化态 U|ψ⟩           期望值 ⟨O⟩
分子构型 R ────[编码]────→ 量子叠加态 ────[量子电路]────→ 纠缠态 ────[测量]────→ 残差预测 Δ
(原子坐标)   角度编码      |ψ⟩ ∈ ℂ^(2^n)    幺正演化U(θ)    Born规则      E_DFT - E_ReaxFF
```

---

## 🔑 三个关键步骤

### 1️⃣ **数据编码**（经典 → 量子）
将分子构型映射到量子态：
$$|\psi(\mathbf{R})\rangle = \bigotimes_{i=1}^{n} R_Y(f_i(\mathbf{R})) R_Z(f_i(\mathbf{R})) |0\rangle$$

- $f_i(\mathbf{R})$：从构型提取的特征（如键长、键角、Coulomb 矩阵元素）
- **作用**：利用 $\sin/\cos$ 引入非线性，映射到高维量子特征空间

### 2️⃣ **量子演化**（变分量子电路）
$$|\phi(\mathbf{R}, \theta)\rangle = U(\theta)|\psi(\mathbf{R})\rangle$$

- $U(\theta)$：参数化量子电路（如 hardware-efficient ansatz）
- $\theta$：可训练参数（类似神经网络的权重）
- **作用**：在量子空间中进行复杂的非线性变换

### 3️⃣ **测量与输出**（量子 → 经典）
$$\Delta(\mathbf{R}, \theta) = \langle\phi|O|\phi\rangle = \sum_m o_m |\langle m|\phi\rangle|^2$$

- Born 规则的 $|\cdot|^2$ 引入第二重非线性
- **输出**：残差能量预测

---

## ⚡ QML 的核心优势
1. **指数维特征空间**：$n$ 个 qubit → $2^n$ 维希尔伯特空间
2. **双重非线性**：编码（$\sin/\cos$）+ 测量（$|\cdot|^2$）
3. **量子纠缠**：捕获特征间的非局域关联

# 第 3 页：为什么 QML 适合力场残差学习？

## 🎯 残差 Δ 的特性与挑战

### ReaxFF 系统误差的分布特征
```
Δ(R) = E_DFT(R) - E_ReaxFF(R)

特点：
• 在平衡构型附近：Δ ≈ 0（ReaxFF 参数化较好）
• 在过渡态区域：Δ 大且高度非线性（键断裂/形成）
• 在强关联区域：Δ 剧烈变化（多电子效应）
• 在异常构型：Δ 不规则（需要外推能力）
```

---

## ✅ QML 的独特优势（vs 经典 ML）

### 1. **数据效率**（小样本学习）
- **问题**：DFT 计算昂贵，训练数据 < 1000 个点
- **经典 GNN**（如 SchNet）：需要 10⁴-10⁶ 个样本才能泛化
- **QML**：
  - 量子核方法在小样本下泛化更好（理论有保证）
  - 量子储备池的高维投影自然正则化

### 2. **非线性表达能力**
经典 ML vs QML 的函数空间：
| 方法 | 非线性来源 | 函数类 |
|------|-----------|--------|
| **多项式回归** | 手工特征 $x^2, x^3, ...$ | 有限阶多项式 |
| **核方法（RBF）** | $e^{-\gamma\\|x-x'\\|^2}$ | 径向基函数 |
| **神经网络** | ReLU/tanh 激活函数 | 分段线性/光滑函数 |
| **QML** | $\sin/\cos$（编码）+ $\|\cdot\|^2$（测量） | **三角函数 + 干涉项** |

**关键**：残差 Δ 在过渡态区域可能有周期性/振荡结构 → QML 的三角函数基天然适配

### 3. **量子纠缠捕获特征关联**
```
经典方法：特征独立处理或通过显式交叉项（x_i · x_j）
QML：纠缠态自动编码高阶特征交互
     |ψ⟩ = Σ α_{i₁i₂...iₙ} |i₁i₂...iₙ⟩
     # 所有 2^n 种组合都有振幅 → 隐式高阶交互
```

### 4. **物理先验易于嵌入**
- 可以设计保持旋转/平移/置换对称性的量子电路
- 可以用物理哈密顿量启发的 ansatz（如 UCCSD-inspired）

## 🔬 理论支持

**定理（Schuld & Killoran, 2019）**：量子核方法在某些函数类上有指数加速的样本复杂度。

**实验证据（Havlicek et al., 2019）**：量子核在合成数据集上优于经典核（小样本情况）。

**我们的假设**：ReaxFF-DFT 残差属于"量子核友好"的函数类（周期性 + 局域关联）。

## 🧮 关键技术选择

### 量子电路设计
```python
# 1. 特征编码层（角度编码）
for i, feature in enumerate(features):
    qc.ry(feature, qubit=i)
    qc.rz(feature, qubit=i)

# 2. 变分层（hardware-efficient ansatz）
for layer in range(L):
    # 单比特旋转
    for i in range(n_qubits):
        qc.ry(θ[layer,i,0], i)
        qc.rz(θ[layer,i,1], i)
    # 纠缠层
    for i in range(n_qubits-1):
        qc.cx(i, i+1)

# 3. 测量层（多个 Pauli 期望值）
observables = [Z_0, Z_1, ..., Z_0⊗Z_1, X_0⊗X_1, ...]
features = [⟨Z_i⟩, ⟨Z_i⊗Z_j⟩, ...]  # M 维量子特征

# 4. 线性回归（经典，快速训练）
Δ_pred = w^T · features
```

### Pauli 分组测量（减少 shots）

# 附录 A：核心公式速查

## 残差学习目标函数
$$\min_{\theta, \mathbf{w}} \sum_{i=1}^{N} \left[\mathbf{w}^T \mathbf{h}(\mathbf{R}_i; \theta) - \Delta_i\right]^2 + \lambda\|\mathbf{w}\|^2$$

其中：
- $\Delta_i = E_{DFT}(\mathbf{R}_i) - E_{ReaxFF}(\mathbf{R}_i)$：真实残差
- $\mathbf{h}(\mathbf{R}_i; \theta) = [\langle O_1 \rangle, \langle O_2 \rangle, ..., \langle O_M \rangle]$：量子特征
- $\langle O_k \rangle = \langle\psi(\mathbf{R}_i)|U^\dagger(\theta) O_k U(\theta)|\psi(\mathbf{R}_i)\rangle$

---

## 梯度计算

### 量子参数梯度（Parameter-shift rule）
$$\frac{\partial \langle O \rangle}{\partial \theta_j} = \frac{1}{2}\left[\langle O \rangle_{\theta_j + \pi/2} - \langle O \rangle_{\theta_j - \pi/2}\right]$$

### 经典权重梯度（解析解）
$$\frac{\partial L}{\partial \mathbf{w}} = 2\mathbf{H}^T(\mathbf{H}\mathbf{w} - \mathbf{\Delta}) + 2\lambda\mathbf{w}$$

其中 $\mathbf{H}_{ij} = h_j(\mathbf{R}_i; \theta)$ 是量子特征矩阵。

## 💡 预期问题 & 回答准备

### Q1: "量子计算机不是还不成熟吗？你怎么用？"
**A**: 我们目前在**经典模拟器**上验证方法可行性。量子电路设计是 NISQ-friendly 的（浅电路、少 qubit），未来可以在 IBM/IonQ 上实验验证。即使在模拟器上，QML 的算法优势（如数据效率）也是有价值的。

### Q2: "QML 比 GNN 好在哪？有理论保证吗？"
**A**: 
1. **数据效率**：量子核方法在小样本下有理论样本复杂度优势（Schuld 2021）
2. **函数类**：QML 的 Fourier 频谱可能更适合残差（周期性/振荡）
3. **实验证据**：Havlicek 2019 在合成数据上验证了优势
4. **我们的假设**：ReaxFF 残差属于 QML 友好的函数类（需实验验证）

### Q3: "训练时间和成本怎么样？"
**A**: 
- **训练**：在模拟器上，500 样本训练约 1-2 小时（vs GNN 的 4-8 小时）
- **推理**：~5 ms/构型（与 GNN 相当）
- **DFT 成本**：与 GNN 相同（都需要 DFT 生成训练数据）

### Q4: "能用于实际 MD 吗？"
**A**: 
- **目标**：准静态模拟（几何优化、过渡态搜索、PES 扫描）
- **不是目标**：纳秒级动力学（仍需经典力场）
- **混合策略**：关键区域用 QML，其余用 ReaxFF

### Q5: "与 Kiss 2022 的区别？"
**A**: 
- **Kiss**：用 QNN 从头学习整个力场（替代 ReaxFF）
- **我们**：用 QML 只学习残差（ReaxFF 作为 baseline）
- **优势**：残差学习数据需求更少、训练更稳定

## 🔬 量子核方法实现

### 如果用量子核（不训练 θ）
```python
# 1. 定义量子特征映射
def quantum_feature_map(x):
    qc = QuantumCircuit(n_qubits)
    # 编码
    for i in range(n_qubits):
        qc.ry(x[i], i)
        qc.rz(x[i], i)
    # 固定的纠缠层
    for i in range(n_qubits-1):
        qc.cx(i, i+1)
    return qc

# 2. 计算量子核矩阵
K[i,j] = |⟨ψ(x_i)|ψ(x_j)⟩|²  # 特征空间内积的模方

# 3. 核岭回归
α = (K + λI)^(-1) Δ_train

# 4. 预测
Δ_new = Σᵢ αᵢ K(x_new, x_i)
```

**优点**：不需要训练量子参数（避免贫瘠高原）
**缺点**：推理需要计算与所有训练点的核（慢）

## 🧠 变分量子电路实现

### 如果用 VQC（训练 θ）
```python
# 1. 量子电路预测函数
def qml_model(x, theta):
    # 编码
    |ψ⟩ = encode(x)
    # 变分层
    |φ⟩ = U(theta)|ψ⟩
    # 测量多个可观测量
    features = [⟨φ|O_k|φ⟩ for O_k in observables]
    return features

# 2. 损失函数
def loss(theta, w):
    predictions = [w @ qml_model(x_i, theta) for x_i in X_train]
    return MSE(predictions, Δ_train) + λ·||w||²

# 3. 联合优化
theta, w = optimize(loss, theta_init, w_init)
```

**优点**：推理快（只需一次量子电路）
**缺点**：训练可能遇到贫瘠高原
