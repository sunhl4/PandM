## 问答日志

### 条目 1

**文献：** H. R. Grimsley *et al.*, *Nat. Commun.* **10**, 3007 (2019)（ADAPT-VQE 奠基）。硬件高效变体见 H. L. Tang *et al.*, *PRX Quantum* **2**, 020310 (2021)（qubit-ADAPT-VQE）。

**标签：** `#ADAPT-VQE` `#算符池` `#VQE` `#自适应线路`

**日期：** 2026-04-07

**问题：**

**ADAPT-VQE**（Adaptive Derivative-Assembled Pseudo-Trotter VQE）通过逐步从算符池中加入「最重要」的激发算符来构造线路。请详细说明其机制；**算符池里通常有哪些算符**、**池大小受什么影响**；在何种情形下 **相对优势**，在何种情形下 **不足或失效**？

**回答：**

---

### （1）算法在做什么（外层 + 内层）

**外层（离散选结构）**：在参考态 $\lvert\mathrm{ref}\rangle$（常为 Hartree–Fock）上已有当前线路 $U$，得到 $\lvert\psi\rangle=U\lvert\mathrm{ref}\rangle$。对池中每个 **厄米生成元** $A_k$，考察若再右乘一层 $e^{i\theta A_k}$ 并在 **$\theta=0$** 处打开，能量

$$
E_{A_k}(\theta)=\langle\psi\vert e^{-i\theta A_k}\,\hat{H}\,e^{i\theta A_k}\vert\psi\rangle
$$

对 $\theta$ 的 **一阶导数**。推导得（见 [ADAPT-VQE.md](./ADAPT-VQE.md) §4）

$$
\left.\frac{d E_{A_k}}{d\theta}\right|_{\theta=0} = i\,\langle\psi\vert[\hat{H},A_k]\vert\psi\rangle.
$$

取 **$\lvert dE/d\theta|_{\theta=0}\rvert$ 最大** 的 $A_{k^\ast}$，把 $e^{i\theta_{\mathrm{new}} A_{k^\ast}}$ **接到**当前 ansatz（再 **内层** 对 **全部** 已有参数 + $\theta_{\mathrm{new}}$ 做普通 VQE）。重复直至所有池元的一阶指标低于阈值 $\varepsilon$，或达到最大轮数。

**内层（连续优化）**：与固定 ansatz 的 VQE 相同（参数移位法则等估计 $\nabla_{\boldsymbol{\theta}}E$）。ADAPT 只负责 **不断增大** 参数与门的集合。

**要点**：外层梯度回答「**下一步该加哪一种激发**」；内层梯度回答「**已选定的多层门，角度怎么调**」。

---

### （2）算符池里「通常」包含哪些算符？

池不是唯一的，文献/代码里常见以下几类（可组合、截断或加对称性筛选）：

| 池类型 | 内容（概念） | 化学 / 实现备注 |
|--------|----------------|-----------------|
| **费米子 SD 池（UCCSD 型）** | 相对 HF 的 **单激发**、**双激发** 对应的 **反厄米** 生成元 $\tau-\tau^\dagger$；映射到 Pauli 串后常整理为 **厄米 $A$** 的 $e^{i\theta A}$ | 与 UCCSD **同一类激发集合**，但 **顺序与子集由 ADAPT 动态决定**，不必一次上全部分量 |
| **更广的费米子池** | 例如 **广义单/双**、或 **更高阶**（三激发…）若显式加入 | 表达力 ↑，池规模 **急剧 ↑** |
| **Qubit 池（qubit-ADAPT）** | 直接在 **Pauli 串**（或稀疏 Pauli 组合）上取生成元，不必先走粒子–空穴图像 | 更贴 **原生门集与比特拓扑**，化学可解释性需自己约束 |
| **约束 / 物理启发池** | 按 **点群、自旋、粒子数** 去掉恒为零或冗余的 $A_k$；或只用 **活性轨道** 子集 | 实际计算 **几乎必须** 做某种截断，否则池太大 |

同一 $A_k$ **选过后是否移出池**、**是否允许重复选**，依实现与变体而定（原版常 **不再重复** 同一生成元）。

---

### （3）池里算符「数量」受什么影响？

**共同规律**：池越大，**单轮外层**要评估的候选越多；**总测量与经典后处理**通常随池大小 **至少线性** 增长（因要对每个 $A_k$ 估 $\langle[\hat{H},A_k]\rangle$ 相关量，而 $[\hat{H},A_k]$ 一般会 **展开成多条 Pauli 串**）。

**主要因素：**

1. **自旋轨道数 $N$（量子比特数）**  
   - **朴素** 费米子双激发：候选数目 **$\mathcal{O}(N^4)$ 量级**（全自旋轨道对）；单激发 **$\mathcal{O}(N^2)$**。  
   - **相对 HF 的粒子–空穴 SD**：双激发常 **$\mathcal{O}\bigl(\eta^2 (N-\eta)^2\bigr)$** 量级（$\eta$：电子数），单激发 **$\mathcal{O}\bigl(\eta(N-\eta)\bigr)$** 量级——仍随体系变大而 **快速膨胀**。

2. **活性空间**  
   若只在 $n_{\mathrm{act}}$ 个空间轨道（或等价自旋轨道子集）上建池，$N$ Effective 替换为 **截断后的轨道数**，池规模 **按上述幂次随 $n_{\mathrm{act}}$ 缩放**。

3. **对称性与筛选**  
   点群、自旋、时间反演等可 **删除** 恒无贡献或线性相关的生成元，**实测池大小** 常小于朴素上界。

4. **池设计选择**  
   Qubit-ADAPT 可选用 **更少但更硬件友好** 的 Pauli 集合，使 **单项数** 与 **编译深度** 的折中不同于费米子 SD 池。

---

### （4）何时相对「优」？

- **大量激发对基态几乎无贡献**（固定 UCCSD 会浪费参数与深度）：ADAPT **按需加长**，最终线路 **往往更短、参数更少**，在多篇小分子基准上体现为 **相同精度下更低深度**。  
- **希望系统地向 FCI（在给定基与活性空间内）逼近**：在 **池已包含所需激发类**、优化可靠的前提下，**贪心加方向** 与 **内层变分** 结合，可 **逐步降低能量**，直至外层一阶条件近似饱和（见 [ADAPT-VQE.md](./ADAPT-VQE.md) §8）。  
- **缓解早期过深 ansatz**：线路从浅到深增长，**早期** 往往比「一步到位深 UCC」更易避免 **部分** 贫瘠高原现象（不保证全局无 BP）。  
- **与对称性约化池联用**：在 **小到中等活性空间** 上，外层筛选 + 内层 VQE 仍是 NISQ 化学中 **常用对照基线**。

---

### （5）何时「劣势」或「不足」？

| 问题 | 说明 |
|------|------|
| **外层测量 / 模拟开销** | **每一轮** 要对 **几乎每个池元** 估梯度；池为 $\mathcal{O}(N^4)$ 时，**真机**上总 shot 与电路批次数可 **非常可观**；经典模拟器上也要反复算对易子期望值。 |
| **$[\hat{H},A_k]$ 展开项多** | 每个 $A_k$ 与分子 $\hat{H}$ 的对易子常分解为 **大量 Pauli 测量项**，进一步放大实验成本。 |
| **贪心非全局最优** | 每步 **局部最优** 的一个 $A_k$ **序列** 未必等于 **最短深度** 或 **最少参数** 的全局最优 ansatz。 |
| **池不完整则必有天花板** | 若池 **不含** 某些激发阶（如只有 SD 没有 **三激发**），一般 **无法** 达到全组态精度；与截断 CI **同类限制**。 |
| **强关联 / 多参考** | 基态可能需 **很多** 激发叠加；ADAPT **仍可能** 需要 **很多轮** 与 **较深** 线路，**根本性成本** 不一定低于其他精心设计的多参考量子方案。 |
| **噪声与有限 shot** | 梯度估计误差会导致 **选错** 算符，**累积** 影响收敛与深度。 |
| **内层优化困难** | 参数增多后，内层仍可能 **局部极小、停滞**；ADAPT 不单独消灭所有 VQE 优化难题。 |
