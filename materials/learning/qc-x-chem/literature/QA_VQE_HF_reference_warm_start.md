### 条目 1

**文献：** （概念专题；与 McArdle *et al.*, *Rev. Mod. Phys.* **92**, 015003 (2020)；Peruzzo *et al.* *Nat. Commun.* **5**, 4213 (2014) 等 VQE / 量子化学综述脉络一致；warm-start 与 UCC 振幅见各实现文档与 Lee *et al.* JCTC 等）

**标签：** `#VQE` `#HF` `#warm-start` `#UCCSD` `#k-UpCCGSD` `#HEA` `#ADAPT-VQE` `#强相关` `#活性空间`

**日期：** 2026-04-14

**问题：**

若以 HF 为参考态做 VQE，是否会因参考态不适用而**不得不增加线路深度**？能否用**经典计算化学**做 warm-up，从而**加快**量子测量与优化上的收敛？

**回答：**

HF 与真实基态在活性空间内差距大时，常同时表现为**需要更强变分族（往往更深或更多 ADAPT 层）**与**更难优化**；但「加深」不是唯一 lever，**CAS/轨道/多参考/嵌入**往往更根本。经典 warm-up（MP2/CCSD 振幅初值、算符池与层序、DMET 等选轨）在实践里**常常**能减少 VQE **外层迭代与总电路评估**；对**单次 shot 的统计方差**主要是**间接**改善。按体系与 ansatz 的细化建议见下文「系统性整理」。

## 系统性整理（正文）

### 1. 问题陈述

- **设定**：量子线路从 HF 参考态出发（常见：Jordan–Wigner / parity 等费米子–量子比特映射后的 HF 计算基态），再堆叠参数化幺正（硬件高效 ansatz、UCC 型、ADAPT 等）最小化能量。
- **疑问 A**：若 HF 参考态「不够用」，是否**不得不**通过**增加线路深度**来补救？
- **疑问 B**：能否用**经典计算化学**先做 warm-up，从而**加快**依赖量子测量/量子评估的优化过程？

### 2. HF 不合适时，与「加深线路」的关系

#### 2.1 机制（直觉）

在多数化学 VQE 叙事里，HF 提供的是**单参考、平均场**起点。若真实基态在该活性空间下具有**显著多参考/强相关**特征，则：

- **表达能力**：同一参数化幺正族要在可行深度内逼近基态，往往需要**更多独立激发通道**或**更强纠缠结构**——在实现上常体现为**更深或更宽的线路**（或 ADAPT 式地**迭代增加许多算符层**）。
- **优化景观**：HF 在参数空间中可能离「好解」很远，易出现**慢收敛、平台、局部极小**；有时表现为「加深也救不回来」，除非同时**改参考、改活性空间、改算符池**。

#### 2.2 结论（精确一点）

- **参考态差** 与 **需要更强 ansatz（常伴随更深）**、**更难优化** 往往**同时**出现。
- 但「加深线路」**不是**唯一 lever：**更合理的活性空间与轨道、多参考预处理、算符池设计** 常常比盲目加深更划算。

### 3. 经典 Warm-up：能加速什么？

#### 3.1 主要加速对象：外层经典优化迭代

经典 warm-up 最直接改善的是 **VQE 外层优化**（基于量子设备/模拟器返回的期望值与可选梯度）：

| 手段 | 作用 |
|------|------|
| 用 **MP2 / CISD / CCSD（或截断）** 振幅初始化 **UCCSD 型**参数 | 把优化起点从「零附近」推到**物理上更可信的簇**附近，常显著**减少迭代步数** |
| **ADAPT-VQE**：用经典梯度/小规模模拟**筛算符池或排序** | 减少无效算符试探，降低**总电路结构更新次数** |
| **DMET / 嵌入 / 轨道选择** 等 | 让 HF 或单参考起点在**化学意义**上更站得住，从源头降低变分负担 |

#### 3.2 对「量子测量」的含义：直接 vs 间接

- **单次测量的统计方差**主要由**可观测量与当前制备态**决定；warm-up **不自动**等价于「每 shot 方差更小」。
- 若 warm-start 使优化更快到达**更低能量的态**，在同样总 shot 预算下，**能量估计的统计表现**往往**间接**更好；且**总评估次数**下降时，**总量子调用量**常明显下降。
- **硬件噪声**下：经典理想参数与**可执行、可标定**线路之间可能有偏移，收益可能被部分抵消；实践中常需**少量再优化**与**误差缓解/标定**。

### 4. 按体系类型：更划算的 Warm-up 组合

下列「强/弱相关」指**在选定活性空间内**的定性判断；同一分子换活性空间结论会变。

#### 4.1 弱相关为主（动态相关占主导、单参考合理）

**典型**：闭壳层有机小分子基态（在合适基组与 CAS 下）、不少键断裂路径若 CAS 已含关键轨道。

**建议**：

- **UCCSD / pUCCD / k-UpCCGSD 等**：优先 **MP2 或 CCSD 振幅** warm-start（与 UCC 参数一一对应或经简单映射时效果最好）。
- **硬件高效 ansatz（HEA）**：经典 warm-start 收益**不稳定**（参数物理意义弱）；可改用**浅层 UCC 预训练**（经典模拟）再映射/蒸馏到 HEA，或直接用 **CISD 系数**启发式初始化（依实现而定）。
- **活性空间**：用 **MP2 自然轨道（MP2 NNO）** 或 **部分占据数** 辅助选轨道，往往比盲目扩大 CAS 更省量子侧成本。

#### 4.2 强相关显著（多参考、近简并、静态相关）

**典型**：某些过渡金属配合物、d–d 相关、部分双自由基、键解离若 CAS 不足等。

**建议**：

- **不要指望**仅靠「更深 UCC from HF」解决定性错误；应优先考虑：
  - **扩大/重选 CAS 与活性轨道**（CASSCF、DMRG-CI 选轨、AVAS 等经典工具）；
  - **多参考量子–经典混合**：如 **qubit-tapering + 多参考初态**、或 **选定的 CASSCF 型初态制备**（若线路与噪声预算允许）；
  - **嵌入（DMET 等）**：把强相关自由度**集中到更小片段**，HF 在片段上的问题会缓和。
- **ADAPT-VQE**：算符池可加入**双激发以外**或**对称性适配**的项；用经典**梯度预筛**可减少量子上的「试错层数」。

#### 4.3 活性空间/轨道明显不当

即使「相关强度」不高，**选错活性轨道**也会让 HF+VQE 表现像「强相关灾难」。

**建议**：先做 **CASSCF / 轨道优化（OO）** 或至少 **MP2 占据分析**，再谈 UCC 深度；这与本仓库中 **OO-VQE / k-UpCCGSD** 等主题常一起出现。

### 5. 按 Ansatz：Warm-up 与「是否必须加深」的对照

#### 5.1 UCCSD（及 Trotter 化 UCC）

- **Warm-up 最对口**：**MP2（cheap）→ CCSD（更准）** 振幅映射到激发参数；文献与实践中**减少 VQE 迭代**的报道较多。
- **深度压力**：Trotter 步数与**项数**带来深度；warm-start **不减少**单项深度，但可减少**外层步数**与无效区域探索。

#### 5.2 k-UpCCGSD / pair 型幺正

- **Warm-up**：可用 **pUCCD / AP1roG** 等 pair 理论或从 **CCSD 振幅投影**到 pair 子空间的思想作初值（依代码接口而定）。
- **与 HF 关系**：对**强相关**体系，pair 型有时比「裸 HF+极深 HEA」更**样本高效**（仍取决于实现与噪声）。

#### 5.3 硬件高效 Ansatz（HEA / Ry–CNOT 链等）

- **经典振幅→参数**没有唯一标准映射；warm-start **收益波动大**。
- **实用策略**：浅层结构化 ansatz 经典预优化、或 **CMA-ES/贝叶斯** 与**好初值**结合；强相关下仍建议回到 **轨道/CAS** 与 **更强物理 ansatz**。

#### 5.4 ADAPT-VQE

- **Warm-up 重点**在 **算符池** 与 **添加顺序**：经典预计算梯度范数、或小体系全模拟排序，可减少**结构更新轮数**。
- **深度**：随迭代**自适应增长**；经典 warm-up 的目标是**少迭代、少废算符**，而非一次给「固定很深」的线路。

### 6. 实施清单（可照此排查）

1. **活性空间是否合理**（轨道与电子数）？不合理则优先经典 CASSCF/选轨，而非加深 UCC。
2. **HF 是否定性正确**？若否，考虑多参考初态或嵌入缩小问题。
3. **Ansatz 与初值是否匹配**？UCC 型优先 MP2/CCSD 振幅；HEA 另寻映射或预训练策略。
4. **噪声与标定**：warm-start 后保留 **10%～30% 步数**做**再微调**；记录**同一噪声模型**下迭代曲线对比（随机初值 vs warm-start）。
5. **度量**：同时看 **总能评估次数**、**到达化学精度能量阈值 ΔE 的步数**、以及 **固定 shot 下的能量方差**。

### 8. 总括（对应原问）

- **HF 参考态不适用**时，确实常表现为**需要更强变分族（往往更深或更多 ADAPT 层）**与**更难优化**；但更应优先审视 **CAS/轨道/多参考**。
- **经典计算化学 warm-up**（振幅初值、算符与层序、嵌入与选轨）在实践里**常常**能加快 VQE **优化收敛**（更少总电路评估）；对**单次 shot 的统计极限**主要是**间接**改善。
- 具体「最划算」组合取决于 **(体系相关强度 × 活性空间质量 × ansatz 可映射性 × 噪声模型)**，上文第 4～5 节按典型情形给出了默认优先级。
