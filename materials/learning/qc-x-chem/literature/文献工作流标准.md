# 文献翻译与问答工作流标准（可修订）

本文件为 **QuantumChemistry** 目录下「论文 PDF → 中文译文 → 问答记录」的约定范本。**后续同类文献处理以本文为准**；你可直接增删改各节，并在文末「修订记录」留一行日期与摘要。

---

## 1. 目录与文件命名

| 类型 | 命名规则 | 示例 |
|------|----------|------|
| 原文 PDF | 与来源一致、便于检索；**无空格** | `2512.13657v2.pdf`、`acs.chemrev.8b00803.pdf` |
| 中文译文 | **英文下划线题名** + `.zh-CN.md`（正文中文） | `Towards_Quantum_Advantage_in_Chemistry.zh-CN.md` |
| 问答记录 | 与译文 **同主干名** + `.QA.zh-CN.md` | `Towards_Quantum_Advantage_in_Chemistry.QA.zh-CN.md` |
| 独立主题问答（非单篇配套） | 自定短名 + `.md` | `QA_波函数集中性.md` |

**原则**

- 译文与问答文件与 PDF **同一目录**（通常为 `QuantumChemistry/`），相对链接可写 `./xxx.zh-CN.md`。
- 文件名用 **ASCII**（英文、数字、下划线），避免空格与特殊符号，便于跨平台与脚本处理。
- 从 PDF / 期刊页无法直接得到英文题名时：可用 **arXiv 号** 或 **期刊缩写_年份_文章编号** 作为主干（如 `2508.02578v2.zh-CN.md`），并在译文题头写清完整中/英文标题。

## 2. 中文译文（`.zh-CN.md`）结构

建议顺序（可按文献类型微调）：

1. **一级标题**：中文题名（可与官方译名或常用译法一致）。
2. **元信息块**：作者、单位（可缩写）、日期、期刊或 **arXiv** 编号等。
3. **分隔线 `---`**
4. **「说明」小节**（固定保留），建议包含：
   - 本文为译文的定位、与原文结构对应关系。
   - **公式渲染**：行内 `$...$`；块公式上下各一行 `$$`，**块末 `$$` 后空一行**再接正文（与现有译文一致）。
   - 参考文献是否保留英文、图表编号是否与原文一致。
   - **指向问答文件**的相对路径与文件名。
5. **正文**：章节编号与原文一致；术语首次出现可加英文或缩写。

**与 PDF 对照**：图表编号、公式 tag、参考文献序号尽量与原文一致，便于答疑时写「见原文 Fig. 2 / 译文 §3」。

## 3. 问答文件（`.QA.zh-CN.md`）结构

**骨架**可直接复制：

`LearningPlan/learning_materials/文献精读问答_模板.md`

**题头必备字段**（与现有一致）：

- 中文书名式标题：`# 《……》问答记录`
- **对应论文**：作者或预印本编号、英文题名、**PDF 文件名**
- **配套译文**：相对链接到 `.zh-CN.md`
- **使用说明**：字段 **文献 / 标签（可选）/ 日期 / 问题 / 回答**；公式规则与译文一致。

**正文区块**：

1. **文章速览**（表格：问题与背景、方法、数据与结论、与课程或项目关联等）— 便于后续提问。
2. **问答日志**：每条复制「条目模板」，按时间追加；可引用译文章节号 `§x.x`。

**与总表同步（可选）**：若你在 `LearningPlan/learning_materials/学习问答记录.md` 中维护总索引，新文献可增加一行或链接到本 `.QA.zh-CN.md`。

## 4. 处理流程清单（给协作者 / Agent）

处理一篇新文献时建议顺序：

1. 确定 **PDF 文件名** 与 **英文下划线主干名**（译文 + 问答共用）。
2. 新建 **译文** `.zh-CN.md`，写好「说明」与问答文件双向引用。
3. 复制 **问答模板** 生成 `.QA.zh-CN.md`，填写题头与文章速览。
4. （可选）在 `学习问答记录.md` 添加索引条目。

---

## 5. 完善本文的方式

- 直接编辑本文件：新增小节、改表格、补充反例或期刊特例。
- 每次重要约定变更，在下方 **修订记录** 增加一行：`YYYY-MM-DD — 摘要`。
- 若希望 Cursor **自动优先**遵循本标准：使用工作区技能 `quantum-literature-zh`（见 `.cursor/skills/quantum-literature-zh/SKILL.md`）；修改流程时同步更新该 SKILL 中「何时读取本文件」的一句话即可。
