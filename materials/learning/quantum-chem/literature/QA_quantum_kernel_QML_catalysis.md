**文献（入门与框架）：** V. Havlíček *et al.*, “Supervised learning with quantum-enhanced feature spaces,” *Nature* **567**, 209 (2019)（量子特征空间 + 量子核与经典 SVM 结合的代表性框架）。综述与局限见 *Nat. Commun.* **15**, 5200 (2024) 等关于 **kernel concentration** 的讨论。

**标签：** `#量子核` `#QML` `#QSVM` `#特征映射` `#催化筛选`

**日期：** 2026-04-07

**问题：**

**量子核方法（QML）**：利用量子特征映射定义分子相似度核，用于催化活性筛选——**具体是怎么实现的**？

**回答：**

---

### （1）先回忆经典「核方法」在做什么

在支持向量机（SVM）等模型里，希望把输入 $\mathbf{x}$（分子描述子）映到某个 **高维特征空间** $\phi(\mathbf{x})$，使得在该空间里 **线性分类/回归** 更容易。显式构造 $\phi$ 往往太贵，于是用 **核技巧**：

$$
K(\mathbf{x}, \mathbf{x}') \;=\; \big\langle \phi(\mathbf{x}),\, \phi(\mathbf{x}') \big\rangle.
$$

只要 $K$ 是 **正定核**（Mercer），经典优化只需 **Gram 矩阵** $K_{ij}=K(\mathbf{x}_i,\mathbf{x}_j)$，不必显式存 $\phi(\mathbf{x})$。

**量子核方法**的核心想法：让 $\phi(\mathbf{x})$ **对应** 量子希尔伯特空间里的一个 **态矢量**（或密度矩阵），用 **量子态重叠** 定义 $K(\mathbf{x},\mathbf{x}')$，从而把「相似度」放进 **指数维**（希尔伯特空间维度随量子比特数指数增长）的 **内积结构** 里，再仍用 **经典** SVM/SVR 做训练与预测。

---

### （2）量子特征映射：从「分子」到「量子态」

**步骤 A — 经典预处理**  

把每个样本（例如一个催化剂构型、吸附构型、或配体）编成 **固定长度实向量** $\mathbf{x}\in\mathbb{R}^d$（或若干离散标签）。常见来源包括：

- **几何**：原子坐标展平、**Coulomb 矩阵**、键长键角、对金属–吸附质的距离等；  
- **拓扑**：分子图特征、配位数、官能团 one-hot；  
- **电子结构粗描述**：形式氧化态、自旋、d 电子数等（筛选管线里常与 DFT 标签联用）。

通常做 **归一化 / 缩放**，使分量适合映射到门参数范围（如旋转角）。

**步骤 B — 参数化量子线路（特征电路）**  

选定 $n$ 个量子比特，构造 **只依赖 $\mathbf{x}$**（训练时不变、换样本才变）的幺正 $U_{\mathrm{enc}}(\mathbf{x})$，使

$$
\lvert\psi(\mathbf{x})\rangle = U_{\mathrm{enc}}(\mathbf{x})\lvert 0\rangle^{\otimes n}.
$$

常见子结构（可组合）：

- **角度编码**：把 $x_j$ 接到 $R_Y$、$R_Z$ 等单比特旋转；  
- **纠缠层**：CNOT / CZ 环、梯子，使不同维度的信息 **非线性混合**；  
- **重复块** $U_{\mathrm{enc}} = W\,E\,W\,E\cdots$（$W$：纠缠，$E$：数据相关旋转），与 Qiskit `ZZFeatureMap`、PennyLane 模板同类。

**要点**：$U_{\mathrm{enc}}$ **不是** VQE 里要训练的那套参数；它由 **数据 $\mathbf{x}$ 唯一确定**。训练发生在下一步的 **核 + SVM**（见 §4）。

---

### （3）量子核：「分子相似度」的数学定义

给定两个样本 $\mathbf{x}$、$\mathbf{x}'$，定义量子态 $\lvert\psi(\mathbf{x})\rangle$、$\lvert\psi(\mathbf{x}')\rangle$。一种 **标准选择**（与 Havlíček *et al.* 框架一致）是 **保真度型核**：

$$
K(\mathbf{x}, \mathbf{x}') \;=\; \big\lvert \langle \psi(\mathbf{x}) \vert \psi(\mathbf{x}') \rangle \big\rvert^2.
$$

即 **希尔伯特空间内积的模平方**，取值在 $[0,1]$：**越接近 1**，两样本在「量子特征空间」里 **方向越重合**，解释为 **相似度越高**。

**在设备上怎么估？**  

- 制备 $\lvert\psi(\mathbf{x})\rangle$，再施加 $U_{\mathrm{enc}}^\dagger(\mathbf{x}')$（或等价 **SWAP 测试** / **Hadamard 测试** 线路），从测量概率读出重叠；  
- 或用模拟器直接计算内积（小规模 benchmark）。

**变体**：不同文献用 **Tr$(\rho_{\mathbf{x}}\rho_{\mathbf{x}'})$**、或 **带投影的核**；催化筛选里 **实现细节以所选库为准**（Qiskit `QuantumKernel`、PennyLane `qml.kernels` 等）。

---

### （4）催化活性筛选：端到端工作流

| 阶段 | 做什么 |
|------|--------|
| **数据** | 收集一批催化剂（或表面位点 + 吸附质）及其 **标签**：活性高低（分类）、TOF/过电位（回归）、或 DFT 算出的描述量 |
| **编码** | 每个样本 → 向量 $\mathbf{x}_i$ → 特征电路 $U_{\mathrm{enc}}(\mathbf{x}_i)$ |
| **核矩阵** | 对所有训练对 $(i,j)$ 计算 $K_{ij}=K(\mathbf{x}_i,\mathbf{x}_j)$（对称正定近似）；$N$ 个样本 **$\mathcal{O}(N^2)$** 次量子评估（可批处理、Nyström 近似降规模） |
| **经典学习** | 用 **预计算核** 训练 **SVM（QSVM）** 或 SVR；**支持向量与拉格朗日乘子**在经典端求解 |
| **预测** | 新分子 $\mathbf{x}$：与训练集（或支持向量）算一列 $K(\mathbf{x},\mathbf{x}_i)$，代入已训练模型得活性类别或标量 |

**与「普通分子机器学习」的差别**仅在于：**Gram 矩阵来自量子线路重叠**，而非多项式核 / RBF / 图核等经典公式。

---

### （5）实现栈（工程上「怎么落地」）

- **Qiskit**：`PauliFeatureMap` / `ZZFeatureMap` 等 + `QuantumKernel` + `SVC(kernel='precomputed')`。  
- **PennyLane**：模板化特征图 + `qml.kernels.square_kernel` 等辅助构造 $|\langle\psi|\psi'\rangle|^2$。  
- **催化场景**：编码层建议与 **对称性**（表面群、配位几何）或 **图结构** 一致，减少无效参数；标签常来自 **高通量 DFT** 或实验，与 [量子计算在计算化学中的方法与文献地图.md](./量子计算在计算化学中的方法与文献地图.md) §4.9 的保守表述一致：**证据链仍在积累**。

---

### （6）优势与主要局限（筛选任务视角）

| 方面 | 说明 |
|------|------|
| **潜在优势** | 特征空间维数随量子比特 **指数** 增长，理论上可表达 **高度非线性** 的相似度；与经典核互补，便于做 **概念验证** 与 **小规模对比实验** |
| **核矩阵成本** | $N$ 个样本需 **$\mathcal{O}(N^2)$** 个核元素，每个元素多次测量 → 数据集一大 **真机很贵** |
| **噪声** | 硬件噪声使 $K_{ij}$ 有偏，Gram 矩阵可能 **半正定破坏**，需正则化或混合经典核 |
| **指数集中（concentration）** | 深度或比特数增加时，$K(\mathbf{x},\mathbf{x}')$ 可能 **趋向常数**，不同分子 **难以区分**——需浅特征图、结构化编码或 covariant kernel 等缓解思路（见近年 *Nat. Commun.* / *Nat. Phys.* 讨论） |
| **标签与泄漏** | 催化标签若全由同一套 DFT 生成，需注意 **与编码特征的信息重叠**（泄漏），交叉验证要按 **催化剂/位点** 分组 |

---

### （7）一句话收束

**实现上**：把每个催化剂样本编成经典向量 $\mathbf{x}$，用 **固定结构的参数化线路** $U_{\mathrm{enc}}(\mathbf{x})$ 得到 $\lvert\psi(\mathbf{x})\rangle$，用 **态重叠的模平方** $K(\mathbf{x},\mathbf{x}')=|\langle\psi(\mathbf{x})|\psi(\mathbf{x}')\rangle|^2$ 作为 **相似度核**，在经典端用 **SVM/SVR** 完成 **活性分类或回归筛选**。
