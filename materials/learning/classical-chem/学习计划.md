## 目标

系统掌握 **经典（非量子计算）量子化学** 的主要近似方法体系，为理解量子计算化学算法（VQE、SQD、QPE、DMET 等）提供 **经典对标基线**——搞清楚量子算法在做什么、替代什么、补充什么。

## 阶段 1：基础（约 2 周）

### 1.1 电子结构问题

- Born-Oppenheimer 近似的内涵与局限
- 电子薛定谔方程 $\hat{H}_e\Psi = E\Psi$
- 精确解的不可行性与电子关联的概念：$E_{\mathrm{corr}} = E_{\mathrm{exact}} - E_{\mathrm{HF}}$

**参考**：详解文档 §1 · [T1] Szabo & Ostlund Ch.1–2

### 1.2 基组

- STO vs GTO；收缩与原始函数
- 基组族：Pople (6-31G 系列)、Dunning (cc-pVXZ)、Ahlrichs (def2)
- 劈裂价、极化函数、弥散函数
- BSSE 与 CBS 外推

**参考**：详解文档 §2 · [T4] Jensen Ch.5

### 1.3 Hartree-Fock

- 平均场近似 → Slater 行列式
- Roothaan-Hall 方程与 SCF 迭代
- RHF / UHF / ROHF
- Koopmans 定理、Brillouin 定理
- 交换空穴与相关空穴
- HF 能抓住什么、漏掉什么

**参考**：详解文档 §3 · [QA_EHF_Hartree-Fock_energy.md](../literature/QA_EHF_Hartree-Fock_energy.md) · [T1] Ch.3

**里程碑**：能用 PySCF 对 H₂O 做一次 HF/cc-pVDZ 计算，解读输出中的 SCF 能量、轨道能、HOMO-LUMO gap。

## 阶段 2：后 HF 波函数方法（约 3 周）

### 2.1 Configuration Interaction (CI)

- FCI 是有限基组内的精确解
- 截断 CI：CIS、CISD、CISDT…
- **大小一致性** 问题与 Davidson 修正
- Selected CI (CIPSI, HCI) —— 现代复兴
- CI 与量子计算的对接：FCI = VQE/QPE 的目标

**参考**：详解文档 §4 · [T1] Ch.4

### 2.2 Many-Body Perturbation Theory (MBPT)

- Rayleigh-Schrödinger 微扰论
- Møller-Plesset 分割 → MP2 公式与 $O(N^5)$ 标度
- MP3、MP4；收敛性问题
- MBPT 何时失效（强关联、近简并）

**参考**：详解文档 §5 · [T1] Ch.6

### 2.3 Coupled Cluster (CC)

- 指数 ansatz $|\Psi\rangle = e^{\hat{T}}|\Phi_0\rangle$
- 为什么用指数（大小广延性）
- CCD → CCSD → CCSD(T) "金标准"
- EOM-CCSD（激发态）
- 计算标度：CCSD $O(N^6)$，(T) $O(N^7)$
- UCC ansatz 与量子计算的联系

**参考**：详解文档 §6 · [T2] Ch.13–14

### 2.4 多参考方法

- 静态关联 vs 动态关联
- CASSCF：活性空间选择策略（AVAS）
- CASPT2、NEVPT2（多参考微扰）
- MRCI
- DMRG 作为"大活性空间"求解器
- 与量子嵌入（DMET + 量子求解器）的衔接

**参考**：详解文档 §7 · [QA_classical_correlation_hard_cases.md](../literature/QA_classical_correlation_hard_cases.md) · [T2] Ch.12

**里程碑**：能对 N₂ 拉伸做 CASSCF(6,6)/cc-pVDZ 计算，画出势能曲线，与 RHF、UHF、CCSD 对比，理解多参考的必要性。

## 阶段 3：密度泛函理论（约 2 周）

### 3.1 DFT 基础

- Hohenberg-Kohn 存在定理与变分定理
- Kohn-Sham ansatz：非相互作用参考系统
- KS 方程与自洽场
- 交换-关联泛函 $E_{\mathrm{xc}}[\rho]$
- 自相互作用误差 (SIE)

**参考**：详解文档 §8 · [T3] Parr & Yang

### 3.2 Jacob's Ladder 泛函分类

| 层级 | 代表泛函 | 关键特征 |
|------|----------|----------|
| LDA | SVWN | 均匀电子气 |
| GGA | PBE, BLYP | 密度梯度 |
| meta-GGA | SCAN, TPSS, r²SCAN | 动能密度 |
| 杂化 | B3LYP, PBE0, HSE | 精确交换 |
| 双杂化 | B2PLYP, DSD-PBEP86 | 精确交换 + MP2 相关 |

色散修正（DFT-D3/D4）与 range-separated hybrid (ωB97X-D, ωB97M-V)

**参考**：详解文档 §9

### 3.3 DFT 实践与局限

- 优势：$O(N^3)$ 成本、良好的几何/热化学
- 局限：SIE、电荷转移、强关联、带隙、范德华力
- DFT+U；TDDFT（激发态）
- 泛函选择实用指南

**参考**：详解文档 §10

**里程碑**：用 PySCF 对苯分子做 B3LYP/def2-TZVP 计算，对比 HF 和 MP2 的结果（几何、能量）；再用 DFT-D3 计算苯二聚体结合能。

## 阶段 4：综合与前沿（约 1 周）

### 4.1 方法层次与决策

- Pople 图（精度 vs 成本，双轴）
- 方法对照表（标度、大小一致性、变分性、典型精度）
- 化学精度（1 kcal/mol）与 benchmark 数据库（GMTKN55、W4-11、S66）
- **决策树**：根据体系特征选择方法

**参考**：详解文档 §11

### 4.2 现代发展趋势（2020–2026）

| 方向 | 核心突破 |
|------|----------|
| DLPNO-CCSD(T) | 近线性标度的金标准精度 |
| Selected CI (CIPSI, HCI, SHCI) | 40–60 轨道活性空间 |
| DMRG 与张量网络 | FeMoco 等多核强关联体系 |
| MLIP（NequIP, MACE） | 量子化学精度 + 力场成本 |
| GFN-xTB 半经验方法 | 千原子构象采样 |
| ML 泛函 (DM21 等) | 神经网络参数化 $E_{\mathrm{xc}}$ |
| 复合方法 (W4, G4) | 系统组合趋近精确极限 |

### 4.3 经典方法与量子计算的衔接

```
经典前端                        量子中间层                    经典后端
─────────────────────────      ──────────────                ────────────
DFT/HF → 轨道 + 积分   ──→   活性空间 + 映射（JW/BK） ──→   1-RDM 回写
AVAS → 活性空间选择     ──→   VQE / SQD / QPE         ──→   片段能量
DMET/嵌入 → bath 构造   ──→   量子求解器              ──→   ML 训练数据
CASSCF → 多参考参考     ──→   UCC ansatz 设计         ──→   DFT 校正
```
