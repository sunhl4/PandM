## 使用约定

1. **每条尽量短**：问题一句话；答案可先写要点，再链到主稿章节（`theory_and_derivations.md` 内标题或 Part 编号）。  
2. **区分类型**：用下面分类打标签，便于检索。  
3. **文献**：需要引用时写 arXiv ID 或 DOI，避免只写作者名。  
4. **与代码对照**：若涉及实现，注明脚本路径（如 `week4_vqc_regression_demo.py`）或 PR/日期。
