# 文献精读笔记（建议目录）

与 [`WEEKLY_LITERATURE_WORKFLOW.md`](../WEEKLY_LITERATURE_WORKFLOW.md) 配合使用。

## 命名建议

- `YYYY-MM-DD_<arxiv_id>_<short_slug>.md`  
  例：`2026-03-30_2401.01234_vqe_measurement_budget.md`

## 内容最小集

1. 论文要解决的核心问题与你主线（计算化学/分子模拟）的关系  
2. 假设与近似（哈密顿量、映射、噪声模型、数据集）  
3. 算法可复现性：缺什么信息就无法复现？  
4. 若复现：计划落在 `benchmarks/` 还是 `quantum_chemistry/tutorials/`？  

**勿将大体积 PDF 提交进 Git**；链接 Zotero 或 arXiv 即可。
