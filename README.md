# PandM — 量子知识图谱

> 工作计划与学习资料的可视化管理系统，量子力学风格交互界面

## 快速启动

```bash
npm install
npm run dev
```

然后打开 http://localhost:5173

## 功能特性

### 可视化
- **力导向网络图** — D3.js 驱动，节点自动布局，物理仿真
- **量子轨道动画** — 节点上的旋转椭圆轨道环（模块级节点）
- **流动粒子** — 连接线上的流动发光粒子
- **量子背景** — Canvas 绘制的粒子场 + 量子干涉波纹

### 交互
| 操作 | 效果 |
|------|------|
| 单击节点 | 打开右侧详情面板 |
| 双击节点 | 聚焦该节点的子图 |
| 滚轮 | 缩放画布 |
| 拖拽背景 | 平移画布 |
| 拖拽节点 | 移动节点位置 |
| `Ctrl+K` | 激活搜索框 |
| `Esc` | 清除搜索 |

### 管理
- **新建分支** — 在任意节点下添加子类别
- **添加资料** — 创建叶子节点，支持 Markdown 内容
- **搜索高亮** — 搜索词高亮匹配节点，非匹配节点淡出
- **持久化** — 所有修改自动保存到浏览器 localStorage
- **重置** — 顶栏重置按钮恢复初始数据

## 技术栈

- **React 18** + **Vite 5** — 构建工具
- **D3.js v7** — 力导向图、缩放、拖拽
- **Framer Motion** — 面板动画
- **Tailwind CSS** — 样式
- **Markdown-it** + **KaTeX** — 节点 Markdown 与数学公式渲染
- **localStorage** — 本地持久化

## 自定义调参（力导向图 / 公式 / 节点样式）

详见 **[docs/可视化与布局调整.md](./docs/可视化与布局调整.md)**：说明要改哪个文件、改哪一段（连线刚度与距离、碰撞半径、斥力、拖拽与缩放、节点 `NODE_TYPE_CONFIG`、KaTeX 与侧栏样式等）。

## 开发

```bash
npm run dev      # 开发服务器 (热更新)
npm run build    # 构建生产版本
npm run preview  # 预览构建结果
```

## 数据结构

节点数据在 `src/data/initialGraph.js` 中定义，支持添加到 localStorage。

重置数据：顶栏 → 刷新图标 → 确认。
